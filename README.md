# Simple Bank System

Консольное приложение для управления банковскими счетами.

## Участники:
- Виктория Б. (https://github.com/ViktorR-BarreL)
- Анастасия Н. (https://github.com/cotlishoc)

## Функционал:
- Создание счета
- Пополнение счета  
- Снятие средств
- Просмотр баланса
- История транзакций
